## ⚙️ 설계 원칙 명문화

### 청산 3각 분업

```
HardStop(ATR×1.6x)  ────── 하방 생명선: 개별 손실 -8% 이내 봉쇄
TrailingStop(10.0%) ────── 추세 추적:  고점 대비 10% 이탈 시 익절
만기청산(18~25일)   ────── 시간 초과:  방향성 없는 횡보 정리
RSI-2≥99(비상구)   ────── 이중 과열:  상승-하락-재상승 혼합 패턴 전용
```

### NaN 모멘텀홀드 (설계적 의도)

```python
# calc_rsi2_series 내 avg_loss=0 시 NaN 반환 → fillna(100) 하지 않음
# 연속 상승(Avg.Loss=0) = [Momentum Hold] 신호
# → RSI-2 exit 비활성화, Trailing Stop에 출구 위임
# → 두산에너빌리티류 멀티배거(+74.7%, +100.9%) 수익 극대화의 근거
```

### L1 Shield (2중 진입 장벽)

```
① m.r ≥ sigThresh  ← 상승/평탄월만 진입 (2026-03 하락월 차단)
② 전월말 KOSPI200 ≥ 5개월 SMA  ← 추세 하락기 차단 (2022년 0건)
```
